# Images for AI Tools

This directory should contain the following images for the AI tools section:

1. `pipecat-ai.jpg` - An image representing the PipeCat AI voice cloning tool
2. `code-assistant.jpg` - An image representing the Code Assistant AI tool
3. `data-analyzer.jpg` - An image representing the Data Analyzer AI tool

## Image Requirements

- Images should be at least 600x400px for good quality display
- Optimal format is JPG for photos or PNG for graphics with transparency
- File size should be optimized for web (under 200KB per image is recommended)

For placeholder usage, you can use stock images or AI-generated images that represent these technologies.
